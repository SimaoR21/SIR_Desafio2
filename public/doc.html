Documentação da API

Esta API oferece as seguintes rotas:

  • GET /students: Lista todos os estudantes.
  • GET /students/:id: Retorna informações de um estudante específico, dado o ID.
  • POST /students: Cria um novo estudante (requer um corpo de requisição com name, course, e year).
  • PUT /students/:id: Atualiza as informações de um estudante existente.
  • DELETE /students/:id: Elimina um estudante específico.
